# Casual Labour Marketplace

## Description
The Casual Labour Marketplace is a web-based platform built with Django that connects casual workers with clients who need short-term services. The system allows different user roles such as clients, workers, and administrators to interact within a centralized platform.

This project is currently under active development.

---

## Features
- User authentication and account management
- Separate roles for:
  - Clients
  - Workers
  - Administrators
- Job posting and management (planned)
- Worker profile management (planned)
- Admin control panel

---

## Tech Stack
- Backend: Django (Python)
- Database: SQLite (development)
- Frontend: Django Templates (HTML, CSS, JavaScript)
- Version Control: Git & GitHub
